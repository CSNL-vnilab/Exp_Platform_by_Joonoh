# Time-bisection (PsychoPy)

Estimation experiment — listener judges whether a tone is "short" or "long" relative to two anchor durations (0.4 s, 1.6 s).

Genre: **estimation**
Framework: **psychopy**
IV (per-trial): `stim_duration` (continuous, sampled from log-uniform 0.4–1.6 s)
DV: response ∈ {short, long}, correct, RT
Setup constants: 6 blocks × 40 trials, fixation 0.5 s, ITI 0.6 s
Saved file: `data/<participant>.csv`
